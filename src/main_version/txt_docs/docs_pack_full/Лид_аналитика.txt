Лидер компетенций - это человек, который занимается развитием компетенций сотрудников определенной профессии.

Лид аналитик занимается онбордингом новых сотрудников на испытательном сроке, постановкой личных целей для сотрудников на квартал, помогает в решении технически сложных задач, декомпозицией крупной задачи на мелкие для оценки, ревью соблюдения технических стандартов/архитектурных принципов, отслеживанием продуктивности сотрудников, проведением обучений.
