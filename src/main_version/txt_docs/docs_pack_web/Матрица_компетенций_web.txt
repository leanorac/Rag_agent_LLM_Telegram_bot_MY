Уровень Junior
Софты: 1.  Желание учиться, подтвержденное делом (Что изучено за последний год? Как это применяется?). 2. Проактивная работа с заказчиком. 3. Умение принимать обратную связь (ОС). 4. Многозадачность.
Харды: 1. Основы HTML и CSS (адаптивная и кроссбраузерная верстка). 2. Основы JavaScript (ES6+). 3. Работа с DOM (добавление, изменение элементов). 4. Понимание работы HTTP (статус-коды, методы GET/POST). 5. Начальные знания Git (клонирование, коммиты, ветки).

Уровень Junior+ Middle-
Софты: 1. Желание учиться, подтвержденное делом (Что изучено за последний год? Как это применяется?). 2. Проактивная работа с заказчиком. 3. Умение принимать ОС. 4. Многозадачность.
Харды: 1. Уверенное владение HTML и CSS (Flexbox, Grid). 2. Уверенное владение JavaScript (Promise, async/await, стрелочные функции). 3. Начальные знания React/Vue/Angular (основы компонентов, состояние). 4. Понимание работы REST API (запросы, ответы, обработка ошибок). 5. Работа с Git (конфликты, слияние, ребейз). 6. Умение работать с DevTools в браузере (отладка, инспектор).

Уровень Middle+
Софты: 1. Желание учиться, подтвержденное делом (Что изучено за последний год? Как это применяется?). 2. Проактивная работа с заказчиком. 3. Умение принимать ОС. 4.  Умение доносить свои мысли до коллег. 5. Умение объяснить заказчику возможные варианты реализации. 6. Многозадачность. 7. Умение выявить у себя недостаток знаний в определенном домене и закрыть его при необходимости.
Харды: 1. Продвинутое владение HTML и CSS (SCSS, CSS-in-JS). 2. Углубленное знание JavaScript (замыкания, контекст вызова, event loop). 3. Опыт работы с React/Vue/Angular (хранилище состояния - Redux, Vuex, Zustand). 4. Умение писать юнит-тесты (Jest, React Testing Library). 5. Работа с WebSocket. 6. Понимание основ CI/CD для фронтенда. 7. Опыт работы с модулями npm/yarn. 8. Настройка линтеров (ESLint, Prettier).

Уровень Senior
Софты: 1. Желание учиться, подтвержденное делом (Что изучено за последний год? Как это применяется?). 2. Проактивная работа с заказчиком. 3. Умение принимать ОС. 4. Умение доносить свои мысли до коллег. 5. Умение объяснить заказчику возможные варианты реализации. 6. Многозадачность. 7. Умение выявить у себя недостаток знаний в определенном домене и закрыть его при необходимости. 8. Понимание влияния своей работы на проект в целом (что критично, что можно отложить). 9. Умение сглаживать напряжение в команде, объяснять приоритеты задач.
Харды: 1. Уверенное владение React/Vue/Angular (SSR, хуки, контекст). 2. Понимание паттернов проектирования (HOC, Compound Components). 3. Опыт оптимизации производительности фронтенда (Lazy Loading, Code Splitting). 4. Опыт работы с TypeScript. 5. Настройка вебпак (Webpack, Vite). 6. Опыт работы с SSR (Next.js/Nuxt.js). 7. Опыт настройки и поддержки кроссбраузерности (Safari, Firefox). 8. Глубокое понимание работы браузера (Event Loop, V8). 9. Опыт работы с монорепозиториями (Nx, Turborepo).

Уровень Lead
Софты: 1. Желание учиться, подтвержденное делом (Что изучено за последний год? Как это применяется?). 2. Проактивная работа с заказчиком. 3. Умение принимать ОС. 4. Умение доносить свои мысли до коллег. 5. Умение объяснить заказчику возможные варианты реализации. 6. Многозадачность. 7. Умение выявить у себя недостаток знаний в определенном домене и закрыть его при необходимости. 8. Понимание влияния своей работы на проект в целом. 9. Умение сглаживать напряжение в команде, объяснять приоритеты задач. 10. Наставничество над коллегами, достижение результата (освоение навыков). 11. Умение давать корректную обратную связь. 12. Умение проводить one-to-one встречи.
Харды: Харды Senior и Lead не отличаются.
Многозадачность - в термин вкладывается НЕ возможность в каждый момент времени думать сразу о нескольких задачах, а возможность переключаться между задачами/проектами (от 2х - оптимально, до 5ти - максимально) без сильной потери эффективности.
